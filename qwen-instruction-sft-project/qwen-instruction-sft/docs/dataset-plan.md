# Dataset Plan

## Versioning

- 当前第一版数据集：`v0.1`
- 目标：先跑通 Qwen 指令微调闭环

## v0.1 Scope

只包含 5 类基础任务：

1. 定义解释
2. 简短问答
3. 改写润色
4. 格式化输出
5. 简单多轮对话

## v0.1 Scale

- 训练集：40 条
- 验证集：10 条

## v0.1 Mix

### 训练集分布

1. 定义解释：12
2. 简短问答：10
3. 改写润色：8
4. 格式化输出：6
5. 简单多轮对话：4

### 验证集分布

保持同样类型分布，但内容不得与训练集重复。

## v0.1 Rules

1. 全部手工或半手工构造
2. 不引入工具调用
3. 不引入长推理链
4. 不引入 Agent 轨迹
5. 不引入 RL 相关样本
6. 所有样本必须格式统一

## v0.1 Success Criteria

1. 能跑通训练
2. 能跑通验证
3. 输出比基座更稳定
4. 格式遵循更好
5. 训练与验证可复现

## Later Phases

### v0.2

- 200 to 500 samples
- 扩展基础任务覆盖
- 开始更系统的 train/eval 对比

### v1.0

- 1k+ samples
- 更完整的任务分布
- 做风格一致性和格式稳定性检查
